What is the use of is Nan function?

  • JavaScript NaN stands for not-a-number.
  • isNaN is a method in JavaScript for use to check a value is a number or not.
  • It returns true if the given value is a NaN else returns false.
  • Syntax:-
    •             < script >
                console.log(isNaN(1)):
                console.log(isNaN("Hello")):
            < / script >
