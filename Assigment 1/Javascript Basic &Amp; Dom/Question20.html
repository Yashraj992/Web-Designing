What is the difference between && and || in JavaScript?

1.&&

  • The && operator evaluates to true if both of its operands are true.
  • If either operand is false, the entire expression evaluates to false.
  • If the first operand is false, the second operand is not evaluated because the overall expression cannot be true regardless of the second operand's value.
    •             true && true //true
            true && false //false
            false && true //false
            false && false //false
  • Example in console

    • 2.||

  • The || operator evaluates to true if at least one of its operands is true.
  • If both operands are false, the entire expression evaluates to false.
  • If the first operand is true, the second operand is not evaluated because the overall expression will be true regardless of the second operand's value.
    •             true || true //true
            true || false //true
            false || true //true
            false || false //false
