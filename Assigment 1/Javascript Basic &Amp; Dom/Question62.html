Check if the string contains a word Script using includes() method?

  • To check if a string contains a specific word, such as "Script," you can use the includes() method in JavaScript.

  • The includes() method determines whether a string contains the characters of a specified string, returning true or false as appropriate.

  • Syntax:
    •             string.includes(searchString, position)
  • searchString: The characters to be searched for within the string.

  • position (optional): The position in the string at which to begin searching. Defaults to 0.
