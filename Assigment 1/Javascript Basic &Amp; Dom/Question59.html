What is Bom vs Dom in JS?

BOM(Browser object model)

  • The BOM provides interaction with the browser.
  • Scope:
  • It includes objects like window, navigator, screen, history, location, and more, which represent the browser and the environment in which the web page is running.
  • Usage:
  • BOM allows developers to control the browser window and communicate with the browser.
  • It includes methods for manipulating the browser window, such as opening new windows, moving and resizing windows, and getting information about the browser itself.
  • Examples:
  • window.alert("Hello World!"); - Shows an alert dialog.
  • location.href - Gets or sets the URL of the current window.

    • DOM(Document object model)

  • The DOM is a programming interface for web documents.
  • Scope:
  • It represents the structure of a document (like an HTML or XML document) as a tree of nodes, where each node represents part of the document.
  • Usage:
  • DOM allows developers to manipulate the content, structure, and style of a web document.
  • It provides methods to traverse the document, add or remove elements, and change the content or style of elements.
  • Examples:
  • document.getElementById("myElement").innerHTML = "Hello World!"; - Changes the content of an element.
  • document.createElement("div"); - Creates a new div element.
  • document.querySelector(".myClass").style.color = "red"; - Changes the color of elements with the class myClass.

    • Difference:

  • Purpose: BOM is concerned with the browser environment, whereas DOM is concerned with the content of the web page.
  • Objects: BOM includes objects like window and navigator, while DOM includes objects that represent document elements like document, element, and node.
