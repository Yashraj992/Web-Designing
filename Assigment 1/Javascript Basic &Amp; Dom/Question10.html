What is Condition Statement?

  • Conditional statements are used to control the flow of a program based on certain conditions.
  • These statements allow the exucution of different code blocks depending on whether a specified condition evalutes to true or false, providing a fudamental mechanism for decision-making in algorithms.
  • There are different types of condition statements.
  • (1)If condition statement:
  • If checks if a condition is a true, if it is, the program executes a block of code.
  • Syntax:
    •             if (condition) {
                code execute if condition is true
            }
  • (2)If else condition statement:
  • If the condition is false, the program excutes the code in the else block.
  • Syntax:
    •             if (condition) {
                code execute if condition is true
            } else {
                code execute if condition is false
            }
  • (3)If-Else if condition statement:
  • The if-else if statement allows for multiple conditions to be checked in sequence.
  • If the if condition is false,the program checks the next else if condition, and so on.
  • Syntax:
    •             if (condition1) {
                code to execute if condition1 is true
            } else if (condition2) {
                code to execute if condition2 is true
            } else {
                code to execute if all conditions are false
            }
  • (4)Switch condition statement:
  • The switch statement is used when you need to check a variable against a series of values.
  • It's often used as a more readable alternative to a long if-else if chain.
  • Syntax:
    •             switch (variable) {
                case value1:
                    code to execute if variable equals value1
                    break;
                case value2:
                    code to execute if variable equals value2
                    break;
                default:
                    code to execute if variable doesn't match any value
            }
