Define a Data Types in js?

  • There are two types of data types in js.
  • (1)Primitive Data Types and (2)Non-Primitive Data Types
  • (1)Primitive Data Types:
  • The predefined data types provided by Javascipt language are known as primitive data types.
  • Primitive also known as in-built data types.
  • Example:
    1. Number
    2. Boolean
    3. Null
    4. Undefined
    5. BigInteger
  • (2)Non-Primitive Data Types:
  • The data types that are derived from primitive data types of the JavaScript language are known as non-primitive data types.
  • It is also known as derived data types or reference data types.
  • Example:
    1. Array
    2. String
    3. Symbol()
    4. Object
    5. Nan
