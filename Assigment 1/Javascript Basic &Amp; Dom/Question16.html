What is the result of the expression (true && 1 && "hello")?

  • true && 1: In JavaScript, the logical AND operator (&&) will return the second operand if the first operand is truthy. Since true is a truthy value, the result of true && 1 is 1.

  • 1 && "hello": Similarly, 1 is a truthy value, so the result of 1 && "hello" is "hello".

  • The final result of the expression (true && 1 && "hello") is "hello".
