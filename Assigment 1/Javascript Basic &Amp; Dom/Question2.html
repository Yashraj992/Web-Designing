How many type of Variable in JavaScript?

  • Variables just like container where we stored an information about any data i.e called variable.
  • Variables is stored information about data i.e called variable.
  • There are three types of variables in JavaScript.
  • (1)Var: var is used to reassigned and redeclare the values.
  • (2)Let: let is used to reassigned but can not redeclared the values.
  • (3)Const: const is used to neither reassigned and neither redeclared the values.
  • We can not change a values inside of constant.
