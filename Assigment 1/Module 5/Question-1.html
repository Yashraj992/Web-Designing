(1)What are the new tags added in HTML5?

Ans:-

  • A lot of new tags added in HTML5. These tags are listed below.

    • Structural or Semantic tags:

    1. article:- Specify a blog, document or newspaper of content in a document.
    2. data:- Machine readable version of the data.
    3. details:- Define any additional information on a topic.
    4. figure:- Specify a self-contained content like photos, diagramas.
    5. footer:- Specify a footer for a scection.
    6. header:- Specify a header for a scection.
    7. nav:- Specify a navigation link in an HTML document.
    8. Section:- Specify a section in an HTML document.
    9. time:- Define a date or time.

    • HTMl5 forms tags:

    1. datalist:- Facilitate an auto complete feature for textfield.
    2. output:- Specify the output of a calculation or an outcome of the user action.

    • Graphics tags

    1. canvas:- Draw canvas in an HTML document.
    2. svg:- Display shapes.

    • HTML5 media tags:-

    1. audio:- Define an audio file in HTML.
    2. embed:- Specify a container for an external file, application.
    3. source:- Specify multiple media resources for a media element.
    4. video:- Specify a video file in HTML.
    5. color:- Define an input field to indicate a color selector.
