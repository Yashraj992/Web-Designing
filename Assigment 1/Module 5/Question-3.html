(3)Semantic element in HTML5?

Ans:-

  • Semantic elements are the elements that describe their meaning to both developer and browser.
  • Many Semantic elements as listed below will be provide by HTML5.
  1. header tag:- Specifies the header or title of the page that will be shown at the top of the page.
  2. nav tag:- Nav tag that is used to build the navbar of the page.
  3. article tag:- Specify a particular block or space on the web page that is independent and has self-contained content.
  4. section tag:- Define the midsection part of the page, which can contain the information about the upcoming events,text contain,etc.
  5. aside tag:- Define the content on the side of the web page.
  6. footer tag:- Mainly used to provide the copyrights or some extra links that are available on the web page.
