What is negative Infinity?

Ans:-

⟹ Negative infinity is a property of the JavaScript Number object.

⟹ Negative infinity is a costant value that is the lowest available.

⟹ This means that no other number is lesser than this value.

⟹ Number.NEGATIVE_INFINITY returns negative infinity.

⟹ Number.NEGATIVE_INFINITY is a number lower than any other number

⟹ Stntax:-

            < script >
                document.getElementById('Id').innerHTML = Number.NEGATIVE_INFINITY;
            </script >
