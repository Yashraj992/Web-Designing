What are undeclared and undefined variables?

Ans:-

⟹ Undeclared variables are those that have not been declared or defined in the current scope, while undefined variables are those that have been declared but not given a value.

⟹ Undeclared - It occurs when a variable which hasn't been declared using var, let or const is being tried to access.

⟹ Syntax:-

            < script >
                let x; 
                console.log(x);
                x = 5;
            </ script >

⟹ Undefined - It occurs when a variable has been declared using var, let or const but isn't given a value.

⟹ Syntax:-

            < script >
                let x; 
                console.log(y);
                x = x*2; 
            </ script >
