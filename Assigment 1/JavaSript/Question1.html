What is JavaScript?

Ans:-

⟹ JavaScript is a Programming Language.

⟹ It is used to make websites and applications dynamic and interactive.

⟹ For example: pop-ups, animations, search bars, buttons, video and audio embeds, drop-down menus and other interactive elements to a webpage.

⟹ You can also use JavaScript for real time updates to webpages without reloading.

⟹ Front-end websites is build on JavaScript like Google, Facebook and Twitter.
