Que-3 What are void elements in HTML? With Example. 

Ans:-
    VOID ELEMENTS
             -A void element is an element in HTML that cannot have any child nodes (i.e., nested elements or text nodes). Void 
             elements only have a start tag; end tags must not be specified for void elements. In HTML, a void element must not have 
             an end tag.

             Ex:- <input type="text"></input> is invalid HTML.
                  The void elements in HTML are as follows: <area> 
                                                            <base>
                                                            <br>
                                                            <col>
