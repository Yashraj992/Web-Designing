Que:-1 Are the HTML tags and elements the same thing? 

  Ans- Both are different element includes starting tag, content and end tag.
  Ending tag not includes content. 
- A tag is a short description of an html object and links.

-Tags have two types 
1.Pair tag 
2.Non pair/void tag

(1) Pair tag:- 
  - Pair tags are tags with both opening and closing. They can have child tags, parent tag and attributes.
  - A pair must be close.
  - For example :- p, div, span, u...

(2) Non pair tag:- 
 - HTML non paired tag if any html tag has both opening and closing tag in a single tag,then such html tag 
 - is known as non paired tag.
 - For example:-br, meta, hr, img, link
                 

                 
ELEMENTS

- HTML element is a component of an html document that tells a web browser how to structure and interpret a part of the html document.
  html element can contain fromatting instruction, semantic meaning , and content

  - Element is a combination of the opening and closing tag,and then the content between them.
    
      - EX:-<start tag>content</end tag>  

- Elements types
  1.Block element  2. Inline element   
  
  (1) Block element:- 
  - block-level element always starts on a new line, and the browsers automatically add some space (a margin) before
   and after the element.
  - block-level element always takes up the full width available (stretches out to the left and right as far as it can).

  Ex:-  <p><div><h1>

(2) Inline element :- 
    - inline element does not start on a new line and  only takes up as much width as necessary This is a <span>
      element inside a paragraph.
      
      
      Ex:- <a>,<b>,<i>,<br>
