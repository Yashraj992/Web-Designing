What are difference between HTML and XHTML?

Ans:- 
            HTML
            (1)HTML stands for Hypertext Markup Language.
            (2)Tags are not extensible.
            (3)HTML is less expressive
            (4)There is no hard rule on the structure of the elements.
            (5)Attribute values are not significant in html.

            XHTML
            (1)HTML stands for Extensible Hypertext Markup Language.
            (2)Tags are extensible.
            (3)XHTML is more expressive compared to HTML.
            (4)The structure of the elements should be followed.
            (5)attributes value are important in XHTML.
