What is the 'class' attribute in HTML? With Example.

Ans:- 
        - The class attribute specifies one or more class names for an element.
        - The class attribute can be used for any HTML element.
        - We can use the same class attribute name with different element.
           Ex:-
           <h2 class="title"> Hello World </h2>
                 <title class="title"> Hello world </title>
