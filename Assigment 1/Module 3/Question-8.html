What are the various formatting tags in HTML?

Ans:- 
            - HTML formatting is a process of formatting text for better look and feel.
            - These tags are used to make text bold, italicized, or underlined.
            
            1.<b>Bold tag
             Bold text
            2.<s>Strong tag
             Bold text
            3.<i>Italic tag
             Italic text
            4.<em>Emphsize tag
             Italic text
            5.<cite>Cite tag
             Italic text
            6.<u>Underline tag
             Used to underline text written between it.
            7.<ins>Ins tag
             Used to underline text written between it.
            8.<del>Delet tag
             Used to display the deleted content.
            9.<strike>Strike tag
             Used to display the deleted content.
            10.<big>Big tag
             Used to increase the font size.
            11.<small>small tag
             Used to increase the font size.
            12.<mark>Mark tag
             Used to highlight text.
            13.<sub>subscripted tag
             It displays the content slightly below the normal line.
            14.<sup>superscripted tag
             It displays the content slightly above the normal line.
            15.<hr>horizontal tag
             Used for creating a horizontal line.
            16.<q>Quated tag
             Creating a short quotations.
            17.<p>Paragraph tag
             Creating a paragraph.
            18.<pre>Preformatted tag
             Preformatted text which is to be presented exactly as written in the HTML file.
            19.<center>Center tag
             Used to set the alignment of text into the center.
            20.<code>Code tag
             Used to represent computer code.
