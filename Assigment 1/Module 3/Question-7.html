What is the difference between the 'id' attribute and the 'class' attribute of HTML elements? With Example.

Ans:-

=>ID:-

ID's are unique

         - Each element can have only one id.
         - Each ID can have an unlimited amount of Styles applied to it.
         - Each page can have only one element with that id.
          Ex:-
          :<h1 id="title"> Question7 </h1>

=>CLASS:-

Classes are not unique

        - You can use the same Class on multiple elements.
        - Class naming is case sensitive.
        - You can use multiple classes on the same element.
         Ex:-
         :<h1 class="title"> Question7 </h1>
         <span class="title"> Question7 </span>
