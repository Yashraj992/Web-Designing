Que-2 What are tags and attributes in HTML? 

Ans:-

-HTML tags
     - HTML tags:- HTML tags are the keywords on a web page that define how your web browser must format and display
       your web page. Almost all tags contain two parts, an opening, and a closing tag.
      
       Ex:-   <html> is the opening  tag and  </html> is the closing tag.

-ATTRIBUTES
         -HTML attribute is a piece of markup language used to adjust the behavior or display of an html element.
         -ATTRIBUTES can be used to change the color,size,or functionality of html elements.
         -ATTRIBUTES are used by including them in an opening html tag 

         Ex:- <tag_name attribute_name="value">content</tag_name>
