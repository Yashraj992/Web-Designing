How can we club two or more rows or columns into a single row or column in an HTML table? With Example.

Ans:- 
             - It can be done by using the rowspan and colspan attribute in HTML.

            =>Rowspan:-
             - The rowspan attribute specifies the number of rows a cell should span vertically.
             - You want to merge two or more Cells in the same column as a single Cell vertically.

              Ex:-

                    Name
