(12)How to maintain the CSS specifications?

Ans:

  • The CSS specifications maintained by the W3C
  • CSS specification provides detailed explanations of the CSS selectors.
  • The CSS Selectors are those, Which selects the HTML elements, on which we want to apply various css style.
  • This CSS specifications also contain the syntex & data type of the CSS style rule.
  • For example Class selector maintained by (.) symbol in internal or external CSS.
  • Id selector maintained by (#) symbol in internal or external CSS.
  • Pseudoclass selector maintained by (:) symbol in CSS.
  • Pseudoelement selector maintained by (::) symbol in CSS.
