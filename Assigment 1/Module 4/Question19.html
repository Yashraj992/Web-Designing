(19)What is the rule set?

Ans:

  • A CSS rule consists of a selector and a declaration block.
  • The selectors points to the HTML element you want to style.
  • The declaration block contains one or more declarations separated by semicolons.
  • Each declaration includes a CSS property name and a value separated by colon.
  • Multiple CSS declarations are separated with semicolons and declaration blocks are surrounded by curly braces.
        Example:-
        p {
            color: red;
            text-align: center;
          }
        p = HTMl elements you want to style
        color and text-align = property
        red and center = property value
