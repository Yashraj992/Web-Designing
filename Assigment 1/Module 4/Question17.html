(17)What is the meaning of the CSS selector?

Ans:

CSS selectors are used to "find" or select the HTML elements you want to style.
  1. Simple selectors
  2. Combinator selectors
  3. Pseudo-class selectors
  4. Pseudo-elements selectors
(1)Simple selectors
Simple selectors select elements by it's name, id & class.
  1. Class selectors:- Class selector apply with (.) and written class name.
  2. Element selectors:- Element selector apply with name of element in style tag in head.
  3. Id selector:- Id selector apply with (#) and written id name.
(2)Combinator selectors
Combinator selector select element based on their relationship between them.
  1. Group selectors:- Group selector, write element in style tag by separated them with (,) symbol.
  2. Child selectors:- Child selector used to apply to css child element of parent element.
  3. General selectors:- General selector used to select the group of element that share the common parent element.
  4. Adjacent selectors:- Adjacent selector is used to select an element that is directly after another specific element.
(3)Pseudo-class selector
Pseudo-class selector apply eith colon(:) symbol.
  1. Link selector:- Link selector is used to apply style on hyperlink.
  2. Hover selector:- Hover selector is used to change style on element, When mouse over them.
  3. Focus selector:- Focus selector is used to change style on element , When mouse over them.
(4)Pseudo-element selector
Pseudo-element selector apply eith colon(::) symbol.
  1. Placeholder selector:- Placeholder selector used to apply css to placeholder attribute. Placeholder is one type of attribute used in creation of login form.
  2. After-Before selector:- After-Before selector in content property must be applied, without content it is dosen't work.
