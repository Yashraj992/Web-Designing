(13)What are the ways to integrate CSS as a web page?

Ans:

(1) CSS stands for the Cascading style sheet.
(2) CSS is used to apply style in website, which is making beautiful and attractive towards client and users.
(3) CSS can be added to HTMl documents in 3 ways:

  1. Inline CSS
  2. Internal CSS
  3. External CSS

1 Inline CSS
  • With inline styles, style sheet information is applied directly to the HTMl element.
  • Inline CSS apply in particular tag/element in style attribute.
  • Instead of defining the style against all instances of an element, you add the style directly to the specific element you want the style to apply to.


    • 2 Internal CSS
  • Internal CSS apply by style tag in head tag.
  • You can use the internal CSS by integrating the (style) element in the (head) section of a HTML web page.


    • 3 External CSS
  • External CSS are the most common method of applying styles to a website.Most modern websites use an external stylesheet to apply sitwide styles to whole pages.
  • To add an external stylesheet to a web page, use the (link)tag, providing the URL of the style sheet in the href attribute, as well as rel=" stylesheet".
