In Bootstrap 4, what is flexbox?

Ans:-

⟹ Bootstrap 4, flex box is used to control the layout and alignment specification of Bootstrap 4 components.

⟹ This box makes it easier to design flexible responsive layout structures without using float or positioning attributes.

⟹ Quickly manage the layout, alignment, and sizing of grid columns, navigation, components, and more with a full suite of responsive flexbox utilities.
