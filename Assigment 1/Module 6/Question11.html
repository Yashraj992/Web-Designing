What are all the looping structure in JavaScipt?

Ans:-

⟹ Loops can execute a block of code a number of times.

⟹ By default loop's starting index is '0'

⟹ Loops are handy, if you want to run the same code over and over again, each time with a different value.

⟹ JavaScript supports different types of loops:

1.For loop - a block of code a number of times

2.For-in loop - the properties of an object

3.For-of loop - the values of an iterable object

4.While loop - a block of code while a specified condition is true

5.Do-while loop - a block of code while a specified condition is true
