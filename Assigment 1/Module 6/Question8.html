What is ===operator?

Ans:-

⟹ === operator is a strict equality comparison operator in JavaScript, which returns false for the values which are not of a similar type.

⟹ This operator performs type casting for equality.

⟹ This operator returns true if both variable contains same information and same data types otherwise return false.

⟹ Example:-

⟹ 5 === 5 (True)

⟹ '5' === 5 (False)
