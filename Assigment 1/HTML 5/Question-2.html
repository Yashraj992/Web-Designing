(2)How to embed audio and video in a webpage?

Ans:-

  • With use of Audio and Video tags , we can embed audio and video in webpage.
  • There are many attributes of <audio> and <video> tag.

  1. autoplay:- When the page is loaded audio/video will be play as soon as possible.
  2. controls:- It displays audio/video control.
  3. loop:- Audio/video start again and again.
  4. muted:- When the page is loaded audio/video will be automatically mute.
  5. src:- URL of the audio and video file.

Audio

  • To embed audio in HTML, we can use the <audio> tag.
  • Before HTML5 audio cannnot added to webpages.
  • Syntax:-
        <audio Controls loop>
            <source src="file name" type="audio file type">
        </audio>

Video

  • To embed video in HTML, we can use the <video> tag.
  • It contains one or more video sources at a time using <source> tag.
  • Syntax:-
            <video Controls loop>
               <source src="file name" type="video file type">
            </video>
