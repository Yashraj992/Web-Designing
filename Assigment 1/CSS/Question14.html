(14)What is embedded style sheets?

Ans:

  • Embedded style sheets allow you to define styles for the whole HTML document in one place.

  • Embedded style sheets refer to when you embed style sheet information into an HTML document using the <style> element.

  • Do this by embedding the style sheet information within <style > <style> tags in the head of your document.

    •             Syntax :
              <head>
              <link rel="stylesheet" href="CSS path">
              </head>
