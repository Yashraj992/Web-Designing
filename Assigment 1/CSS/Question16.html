(16)What are advantages and disadvantages of using external style sheets?

Ans:

(1)Advantages of using External style sheets
  • With the help of External style sheets, the styles of numerous documents can be organized from one sibgle file.
  • In External style sheets, Classes can be made for use on numerous HTML element types in many forms of the site.
  • Since the CSS code is in a separate document, your HTML files will have a cleaner structure and are smaller in size.
  • You can use the same .css file for multiple pages.
(2)Disadvantages of using External style sheets
  • An extra download is essential to import style information for each file.
  • Your pages may not be rendered correctly until the external CSS is loaded.
  • Uploading or linking to multiple CSS file can increase your site's download.
