(18)What are the media types allowed by CSS?

Ans:

One of the most important features of style sheets is that they specify how a document is to be presented on different media: on the screen, on paper, with a speech,etc. We have four ways to specify media types for style sheets-
  1. all:- Used for all media type devices.
  2. Print:- Used for printers
  3. Screen:- Used for computer screens, tablets, smart-phones etc.
  4. Speech:- Used for scrrenreaders that "reads" yhe page out loud.
One way to use media queries is to have an alternate CSS section right inside your style sheet.

Example: 
        @media screen and (min-width:500px)
          {
            body{
                 background-color: red
                }
            
          }
